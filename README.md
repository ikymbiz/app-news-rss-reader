# ⬛️ News RSS Reader

ブラウザ上で完結する、AI駆動・サーバーレスのモダンRSSリーダー。
「Obsidian Terminal」デザインシステムを採用した美しいダークUIと、モバイルライクな直感的スワイプ操作、そして複数のLLMを組み合わせた独自の「AIパイプライン機能」を備えています。

## ✨ 主な機能

### 📱 直感的なスワイプアクション
スマートフォンアプリのような操作感をブラウザで実現しました。
- **右スワイプ**: 記事を「お気に入り」に追加（緑色ハイライト）
- **左スワイプ**: 記事を論理削除（赤色ハイライト）。削除された記事は一覧から消え、3日経過後にデータベースから自動的に物理削除されます。

### 🤖 強力なAIタスクパイプライン
Gemini, OpenAI (ChatGPT), Anthropic (Claude), xAI (Grok) の主要なLLMをサポート。
取得した記事に対して、ユーザーが自由に定義したAIプロンプトを直列（チェーン）で実行できます。
- 記事が自分の興味（例：AI関連技術）に合致するか自動判定し、タグ付け。
- 長文の要約作成や、ネガポジ判定などをバックグラウンドで自動処理。

### 🌐 DeepL ワンクリック翻訳
DeepL APIと連携し、英語などの外国語記事のタイトルと概要をワンクリックで日本語に翻訳します。

### 🔒 完全なローカル＆プライバシーファースト
バックエンドサーバーは不要です。
設定データ、取得した記事、APIキーはすべてブラウザの **IndexedDB** に安全にローカル保存されます。外部にデータが漏れることはありません。

### 📁 ファイルシステム同期・データ管理
- **File System Access API** をサポート。指定したローカルのJSONファイルにデータベースを自動同期・バックアップします。
- 過去の記事データのエクスポート/インポート機能も標準搭載。

## 🚀 使い方

本アプリケーションは、**単一のHTMLファイル**で構成されています。Webサーバーを用意する必要も、npm install を行う必要もありません。

### 1. 起動方法
1. `index.html` をダウンロードします。
2. Google Chrome, Microsoft Edge, Safari などのモダンブラウザで `index.html` を開きます。

### 2. 初期設定
1. 右上の **設定アイコン (⚙️)** をクリックして設定画面を開きます。
2. **RSSフィードの追加**: 購読したいメディアやブログのRSS URLを入力し、「追加」ボタンを押します。
3. **APIキーの登録 (任意)**: AI機能や翻訳機能を使用する場合は、お持ちのAPIキーを入力し「キーを保存」を押してください。（※キーはブラウザ内部にのみ保存されます）

### 3. 記事の取得と閲覧
1. ヘッダーの **更新ボタン (🔄)** をクリックすると、登録したフィードから最新記事を取得します。
2. スワイプによる仕分け、ワンクリック翻訳などを活用して記事を消化します。

## 🧠 AIパイプラインの設定方法

設定画面の「AIフィルタリング パイプライン」機能を使用すると、記事ごとに自動処理タスクを走らせることができます。

タスクのプロンプト内では、以下の特別なプレースホルダー変数を使用できます：
- `{{TITLE}}` : 記事のタイトルに置換されます
- `{{CONTENT}}` : 記事の概要（要約テキスト）に置換されます
- `{{PREV_RESULT}}` : 1つ前のAIタスクの出力結果に置換されます。これにより、「タスクAで抽出したキーワードを、タスクBで日本語に翻訳する」といった**プロンプトチェーン**が可能になります。

**設定例：AI関連度判定**
> 以下の記事はAI、機械学習、LLM、または関連技術について記述していますか？
> 回答は「AI関連: true」または「AI関連: false」から始め、理由を1文で述べてください。
> 
> タイトル: {{TITLE}}
> 概要: {{CONTENT}}

## 🖥 開発者向け情報

### UI/UXデザイン
本アプリは `:root` に定義されたデザイントークンにより管理されています。
テーマカラーの変更を行いたい場合は、CSSの `/* ── 1. Color Tokens ── */` セクションのHEXコードを変更するだけで、全体のデザインを一括変更できます。

### 制限事項・注意事項
- ブラウザのセキュリティ制約により、他ドメインのRSSを取得するために `api.allorigins.win` をプロキシとして使用しています。大量のアクセスを行う場合は、自前のプロキシサーバー（CORS Anywhereなど）への変更をご検討ください。
- 「File System Access API」は、一部のブラウザ（Firefoxなど）では完全にサポートされていない場合があります。その場合は手動でのインポート/エクスポート機能をご利用ください。
